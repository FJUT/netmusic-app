##目前实现功能

1. 用户歌单
2. 歌单详情
3. FM
4. 音乐播放（暂停，上下一首，歌词，快进快退）
5. 评论显示
6. MV
7. 专辑页
8. 歌手页
9. 登录
10. 歌曲红心,FM trash
11.收听记录
12.歌单歌曲推荐
